PC端hexo框架搭建个人博客

安装hexo框架

1.安装node.js和git

2.安装hexo框架

在cmd窗口使用npm进行安装

npm install -g hexo-cli

3.使用hexo -v来验证是否安装成功

hexo简单使用

hexo命令 hexo命令简写 hexo命令作用
hexo init 新建一个网站，将一个文件夹初始化
hexo server hexo s 开启hexo服务
hexo deploy hexo d 部署博客到服务器
hexo generate hexo g 将博客内容进行编译，生成静态页
hexo new “博客名字” 无 用于新建博客
hexo new page name 无 新建一个博客管理页
hexo clean 无 清除生成的数据

将博客发布到GitHub上

1.在GitHub上新建一个仓库（repository）,命名必须符合规范

命名规范：账户名.github.io

2.客户端安装工具hexo-deployer-git,用于推送博客到github仓库。

npm install –save hexo-deployer-git

3.使用hexo编译生成博客

4.使用hexo d 进行发布

5.访问 网址 ：账户名.github.io 即可

博客主题更换

在github上找到hexo主题，按照要求进行配置即可
